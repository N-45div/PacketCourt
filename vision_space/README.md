# PacketCourt Vision

Private ZeroGPU companion for PacketCourt. It uses OpenBMB MiniCPM-V-4.6 to
transcribe only visible evidence from front and back food-package photos.

The main PacketCourt product applies deterministic claim auditing, arithmetic,
and refusal rules after transcription.
